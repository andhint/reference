# Reference

## Virtual Envs using venv

1) Creating 

`python3 -m [env name]` 
`python3 -m venv`

2) Activating

`source [env name]/bin/activate`
`source venv/bin/activate`

3) Deactivating

`deactivate`

4) Removing

`rm -R [env name]`
`rm -R venv`

Good practice to make environment named `venv` within each project directory
